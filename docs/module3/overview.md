# Module 3: The AI-Robot Brain (NVIDIA Isaac™)

## Introduction

NVIDIA Isaac is the AI platform that brings intelligence to robots. It combines simulation, perception, manipulation, and navigation into a unified framework optimized for real-time performance.

## Module Focus

**Advanced perception and training** - Learn to use cutting-edge AI tools for robot intelligence.

## What You'll Learn

### NVIDIA Isaac Ecosystem

1. **Isaac Sim**
   - Photorealistic simulation
   - Synthetic data generation
   - Physics-accurate environments
   - Multi-robot scenarios

2. **Isaac ROS**
   - Hardware-accelerated perception
   - VSLAM (Visual SLAM)
   - Object detection
   - Depth processing

3. **Nav2 Integration**
   - Path planning for bipedal robots
   - Obstacle avoidance
   - Behavior trees
   - Recovery behaviors
