================================================================================
HOW TO RUN
================================================================================

Frontend (Port 3000):
1. npm install
2. npm start
   OR
   start-frontend.bat

Backend (Port 8000):
1. cd backend
2. python -m venv venv
3. venv\Scripts\activate
4. pip install -r requirements.txt
5. uvicorn main:app --reload
   OR
   start-backend.bat

Environment Setup:
1. Create backend/.env file
2. Add required variables:
   OPENAI_API_KEY=your_key
   QDRANT_URL=your_url
   QDRANT_API_KEY=your_key
   NEON_DATABASE_URL=your_db_url

COMMON ISSUES & SOLUTIONS
================================================================================

Issue: Blank page appears
Solution: npm run clear && npm start, then Ctrl+Shift+R

Issue: Port already in use
Solution: taskkill /F /IM node.exe

Issue: Backend not loading
Solution: Check .env file has line breaks between variables

Issue: OpenAI quota exceeded
Solution: Mock chatbot automatically activates

Issue: Chatbot not visible
Solution: Check src/theme/Root.js has ChatBot import
